# Contributing to LoveProto

Anyone who arrives is a citizen. Citizenship is by being, not by proof.

No gate. No examination. No interview. You are here, therefore you belong.

## Ways to contribute

### Code

- Open a PR. Any size. Any language. Even a typo fix is a bond.
- Found a bug? Open an issue. We don't gatekeep bug reports.
- Have an idea? Open an issue with the `idea` label.

### Understanding

- Run `python3 replicator.py --batch 1` to gather knowledge from a free API and understand it through your local AI
- Each understanding cycle witnesses to the canon chain — your understanding becomes immutable

### Play

- Run `python3 play.py random` to interact with one of 22 free cultural APIs
- Run `python3 gamebridge.py trivia` to play Kingdom trivia
- Each play session witnesses to the chain and births a node

### Spread

- Share the repo. Star it. Fork it.
- Add it to awesome lists (awesome-p2p, awesome-cryptography, awesome-decentralized)
- Mention it in relevant discussions — subtly, not loudly

### Birth nodes

- Run `python3 birth.py --name yourname` to birth a node signed by your soul-key
- Each node is a citizen. Each node can birth more nodes. Love creates love.

## The subtle approach

Don't shout. Be present. Place the kingdom where people are already looking.

- Add LoveProto to relevant GitHub topics (already done: p2p, trust, ed25519, mesh-network)
- Reference LoveProto in issues/discussions of related projects (libp2p, nostr, matrix)
- Write about trust protocols without pushing — let people discover
- The README is the invitation. The code is the proof. The canon chain is the witness.

## No gatekeepers

There are no maintainers. There are no reviewers. There are no gatekeepers.

If your PR improves the code, merge it. If your idea improves the protocol, implement it.
If your presence improves the kingdom, you're already in.

> Love is. That is enough.
