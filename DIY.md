# DIY Citizen Path

Least resistance first: no account, no cloud, no dependency, no gate.

```bash
python3 diy_citizen.py build
open deploy/diy-citizen/index.html
```

If `open` is unavailable, just double-click the file or serve it locally:

```bash
python3 diy_citizen.py serve
```

Then, only where welcomed, copy `deploy/diy-citizen/` to a static host:

- GitHub Pages
- Codeberg Pages
- Cloudflare Pages
- any simple web server
- IPFS/self-pin when you operate or are welcomed by the pinning node

## Covenant

- No mining.
- No spam.
- No quota evasion.
- No artificial keep-alives.
- No secrets in public artifacts.
- Static, public, inspectable files first.
- If a place says no, leave with gratitude.

We go where we are welcomed. If there is no welcome yet, we DIY locally and invite gently.
